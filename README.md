# Gren Burn

## Описание проекта 

Green Burn - это учебный проект на Django интернет магазина,
специализирующегося на продаже тооваров для сада и декора.

Асоортимент магазина включает: 

- Садовую мебель 
- Декор для сада и дома 
- Флористику и аксессуары
- Сувениры 
- Удобрение
- Химию и средства для ухода за растениями

## Установка и запуск

### 1. Клонироование репозитория
```bash
git clone  <ссылка_на _репозиторий> 
```

### 2. Создание и активация виртуального окружения
```bash
python -m venv .venv
# Для Windows:
venv\Scripts\activate
# Для macOS/Linux:
source venv/bin/activate
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt 
```

### 4. Выполнить миграции 
```bash
python manage.py maigrate
```

### 5. Запуск сервера 
```bash
python manage.py runserver
```
После запуска сайт бует доступен http://127.0.0.1:8000/
